# Contributing

Thanks for contributing to Ducky Pwn Docs!

## Setup
- Install dependencies: `npm install`
- Run dev: `npm run electron:dev`

## Guidelines
- Keep UI changes consistent with the existing visual language.
- Prefer small, focused PRs.
- Include screenshots for UI changes.

## Reporting Issues
Please include:
- OS and app version
- Steps to reproduce
- Expected vs actual behavior
- Logs if available
